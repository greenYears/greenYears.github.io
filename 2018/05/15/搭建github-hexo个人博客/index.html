搭建github+hexo个人博客

发表于 2018-05-15 | 分类于 工具

开始之前

很久之前就一直想自己搭建一个个人博客系统，但是均因为种种原因而搁浅。这次算是一个全新的开始吧。开始记录生活，学习中的值得记录的点滴吧。

github+hexo的搭建流程网上有很多详细的图文教程，第一篇文章也不知道记录点什么，就拿这次的搭建过程来抛砖引玉了。此处只是粗略列举下步骤。

正式开始搭建

github上的工作

  1. 登录github帐号，此处略过。
  2. 创建一个新的repository。点击如下图
  3. 初始化你的repository。建议Repository name填写内容类似:xxx.github.io;其中xxx是你在github上的名字。这样就可以直接使用https:xxx.github.io访问你的主页了。

本地开发环境搭建

  1. 开发电脑一台，windows,mac等，本人是在mac上开发的。
  2. git。
  3. node，npm。可以使用nvm来管理你本地的node环境版本，还是挺方便的。

开始搭建hexo

  1. 搭建开始可以参考官网;
  2. 本地debug: hexo s --debug
  3. 打包上传: hexo clean && hexo g && hexo d
